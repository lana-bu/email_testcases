# email_testcases
This project is for my Assignment #3 in CIS 376 (Software Engineering II) at UM-Dearborn. The Apache email project was imported in, but I created unit tests for 10 methods of the Email class. I developed the unit tests to get familiar with making JUnit test cases that cover the majority of the lines of code for a function. The goal for my test cases was to get at least 70% test coverage for each of the 10 Email functions.
